Proyecto de Automatización de Telescopios

Este proyecto consiste en el desarrollo de un sistema de control informatizado para telescopios motorizados utilizando una placa ESP32, drivers DRV8825 y motores paso a paso NEMA 17. El objetivo es permitir que el telescopio pueda orientarse automáticamente hacia objetos astronómicos seleccionados desde una interfaz digital.

Objetivo del Proyecto

Desarrollar un sistema de automatización accesible, intuitivo y de bajo costo que permita controlar el movimiento de un telescopio en los ejes de azimut y altituf.

Componentes Principales

  • Placa ESP32
  • Drivers de motor DRV8825
  • Motores paso a paso NEMA 17
  • Interfaz digital de control
  • Soportes impresos en 3D

Funcionamiento

El sistema recibe comandos desde una aplicación o software compatible. La ESP32 interpreta esos comandos y controla los motores paso a paso con precisión a través de los drivers DRV8825. De esta forma, el telescopio se orienta automáticamente hacia el objeto elegido sin necesidad de operación manual.

Ventajas del Sistema

  • Control remoto y automatizado del telescopio.
  • Mayor precisión en el seguimiento de objetos.
  • Permite aprendizaje y ampliaciones futuras.
